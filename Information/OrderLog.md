## Описание класса OrderLog
**OrderLog** - класс, представляющий собой журнал изменений заказа
# Атрибуты
- **ID**: int
- **Date**: DateTime
- **Event**: string
- **Order**: Order
# Описание атрибутов
- **ID** - идентификатор в БД
- **Date** - дата изменения
- **Event** - причина изменения
- **Order** - заказ, в котором произошли изменения
